# TaxTrace AI - e-Invoicing Compliance System

AI-powered invoice validation and compliance system for LHDN MyInvois.

## Features

- 📊 **Dashboard** - Real-time compliance overview
- 📄 **Invoice Queue** - Manage incoming invoices
- 🤖 **AI Analysis** - Claude-powered PDF extraction
- 🔍 **Discrepancy Detection** - Cross-validation with LHDN MyInvois
- 📧 **Resolution Comms** - AI-drafted vendor communications
- 📈 **Analytics** - Compliance trends and insights
- 📝 **Audit Log** - Immutable record of all actions

## Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite (with SQLAlchemy)
- **Frontend:** HTML5, CSS3, Vanilla JS
- **AI:** Anthropic Claude API
- **Charts:** Chart.js
